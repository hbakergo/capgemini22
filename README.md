<h1>DESAFIO DE PROGRAMAÇÃO CAPGEMINI</h1>

<h2>Tecnologias utilizadas:</h2>
->Spring Tool Suite 4 (STS)<br>
->Java 11<br>
->Junit 4<br>


<h2>==============QUESTÃO 01==============</h2>

<h3>Instruções para rodar QUESTÃO 01</h3>

<b>-> LINHA DE COMANDO:</b>
1) acessar o diretório \bin na pasta do projeto;
2) digitar java Questao01, teclar ENTER

<b>-> DENTRO DO STS:</b>
1) File -> Open Projects from file system...
2) Clicar em Directory
3) Escolher o diretório do projeto
4) Clicar em Finish
5) Clicar no menu Run -> Run

<h3>Instruções para usar QUESTÃO 01</h3>
1) Digitar o número de degraus da escada e teclar ENTER

OBS: No STS caso seja necessário avaliar/testar os "testes unitários", deverá abrir a classe LadderTest e clicar no menu Run -> Run as -> JUnit Test
